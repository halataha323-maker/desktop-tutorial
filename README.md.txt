# AI Agent for Gmail & Google Calendar Management

פרויקט זה מציג סוכן בינה מלאכותית אוטונומי המסוגל לקרוא מיילים בשפה חופשית, לקבוע פגישות ביומן גוגל במידה והזמן פנוי, או להשיב במייל דחייה במידה והזמן תפוס.

## איך מפעילים את הפרויקט?
1. להתקין את כלי הניהול `uv`.
2. לוודא שקובץ `credentials.json` נמצא בתיקיית השורש.
3. להריץ את הפקודה: `uv sync` להתקנת כל התלויות.
4. להריץ את הסוכן: `uv run main.py`.
